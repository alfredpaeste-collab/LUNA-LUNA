LUNA GUIDE
Learn how to use Luna & set up your API keys
◈ GETTING STARTED
1
Sign in or continue as guest
When you first open Luna, enter a display name to start chatting. No account required — your chat history is saved locally on your device.
2
Type your message and send
Click the input bar at the bottom and type anything. Press Enter to send, or Shift+Enter for a new line. Luna responds in real-time with streaming text.
3
Add your own API key (optional but recommended)
Luna uses shared API keys by default, which have rate limits. Adding your own free Groq key gives you a personal quota. Go to Settings → Personal API Key and paste it in.
4
Upload images or files
Drag and drop an image or file onto the chat, or click the 📎 attachment button. Luna can describe images, read documents, and answer questions about uploaded content.
5
Use voice input
Tap the 🎤 microphone button to speak your message. Luna transcribes your voice and sends it automatically. Works on both desktop and mobile.

◈ WHY ADD YOUR OWN KEY?
!
Avoid "all providers unavailable" errors
Luna shares API keys among all users by default. When too many people chat at once, the shared keys hit their rate limit. Adding your own free key gives you a personal quota that nobody else uses. All keys below are 100% free.
◈ PROVIDER OPTIONS
🟣 Groq FASTEST FREE 14,400 req/day
Luna's primary provider. Groq uses dedicated LPU hardware for ultra-fast responses — typically 3–5× faster than other providers. The free tier is very generous.
1
Go to console.groq.com/keys and sign up for free
2
Click Create API Key, give it a name, then copy it
3
In Luna, open Settings → Personal API Key and paste it
Format: gsk_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
🔵 Google Gemini FALLBACK #1 FREE 15 RPM · 1M TPM
Luna's first fallback when Groq is busy. Powered by Google's Gemini 2.0 Flash model — fast and capable. The free tier includes 1 million tokens per minute.
1
Visit aistudio.google.com/app/apikey (Google account required)
2
Click Create API key and copy it
3
Open script.js and paste it as the value of GEMINI_API_KEY
Format: AIzaSy_xxxxxxxxxxxxxxxxxxxxxxxxxxxx
🟠 OpenRouter FALLBACK #2 FREE Free models available
Luna's final fallback. OpenRouter routes to many free models (Llama 3.3 70B, Mistral 7B, Gemma 2, and more). Great as a last resort when other providers are busy.
1
Go to openrouter.ai/keys and create a free account
2
Click Create Key and copy it
3
Open script.js and paste it as the value of OPENROUTER_API_KEY
Format: sk-or-v1-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
◈ POWER USER TIPS
⚡
Multiple Groq Keys
Add up to 5 Groq API keys in API_KEY_POOL inside script.js. Luna round-robins them so your quota multiplies by the number of keys.
🎨
Switch Themes
Luna has 4 themes — Neural (purple), Astral (blue), Solar (amber), and Galactic (pink). Switch from the bottom of the sidebar to match your vibe.
📌
Pin & React to Messages
Hover (or long-press on mobile) over any message to see options: react with emoji, copy the text, pin it to the top of your chat, or regenerate the response.
📱
Install as App
Use the Add to Home Screen button in the sidebar to install Luna as a PWA. It works offline for reading history and launches like a native app.
🔍
Search Your History
Press Ctrl+K or click Search in the sidebar to instantly find any past message by keyword.
📚
Study Game
Ask Luna to explain a topic, then open the Study Game from the sidebar. Luna turns the explanation into interactive flashcards to help you memorize it.
🌐
Auto Provider Fallback
If Groq is rate-limited, Luna automatically switches to Gemini, then OpenRouter — no action needed from you. Add all 3 keys for maximum uptime.
💾
Export Your Chat
Use the Export button (in the sidebar or keyboard shortcuts panel) to save your entire conversation as a formatted HTML file you can keep or share.
